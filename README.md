**Este proyecto es una calculadora web desarrollada con HTML5, CSS3 y JavaScript (ES5).
Permite realizar operaciones matemáticas básicas y guarda un historial de los cálculos realizados utilizando localStorage.**

Funcionalidades:

- Suma, resta, multiplicación y división
- Interfaz moderna con diseño oscuro estilo iPhone
- Historial de operaciones persistente
- Opción para borrar el historial
- Compatible con navegadores modernos
